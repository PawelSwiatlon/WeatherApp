Weather application created by Paweł Światłoń, the application uses information provided through "Open Weather Api".

About api

Open Weather Api is one of the leading digital weather information providers.
Weather data is open: current weather, forecasts, wind, clouds, data from weather stations are available through APIs, maps and other products.

Site Info
